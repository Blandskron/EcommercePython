## Cómo Empezar

Para obtener una copia local del proyecto, sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/Blandskron/EcommercePython.git
    ```
2.  **Accede al directorio:**
    ```bash
    cd EcommercePython/
    ```
