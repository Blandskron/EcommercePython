# UX/UI Designer

## ¿Qué es?

El UX/UI Designer es quien se encarga de diseñar productos digitales pensando en el usuario.

- UX: se enfoca en que el sistema sea fácil de usar
- UI: se enfoca en cómo se ve (diseño visual)

---

## Objetivo

Hacer que una página o aplicación sea:
- Fácil de entender
- Rápida de usar
- Visualmente clara

---

## Funciones principales

### UX
- Investigar qué necesitan los usuarios
- Definir cómo van a navegar dentro del sistema
- Hacer wireframes (estructura básica)
- Detectar problemas de uso

### UI
- Diseñar la interfaz (colores, botones, tipografía)
- Mantener orden visual
- Adaptar el diseño a celular y computador

---

## Herramientas

- Figma
- Adobe XD
- Sketch

---

## Conocimientos técnicos

- HTML y CSS (básico)
- Diseño responsive
- Usabilidad
- Accesibilidad

---

## Flujo de trabajo

1. Entender el problema
2. Pensar solución
3. Diseñar estructura (wireframe)
4. Diseño visual
5. Probar y mejorar

---

## Ejemplo

En un e-commerce:

- UX: hace que comprar sea rápido y simple
- UI: diseña botones y estructura para que se vea claro

---

## Conclusión

El UX/UI Designer es importante porque mejora la experiencia del usuario y hace que los sistemas sean más fáciles de usar.
