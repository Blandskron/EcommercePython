# Perfil del Cargo: Scrum Master

**Responsable:** Patricio Saavedra  
**Rama de trabajo:** `sm/patricio`

## Resumen del Rol
Líder de servicio encargado de asegurar que el equipo de TI comprenda y aplique Scrum. Actúa como facilitador de procesos y eliminador de impedimentos para maximizar la entrega de valor.

## Responsabilidades Específicas
1. **Facilitación:** Moderar Daily Scrums, Sprint Plannings y Retrospectivas.
2. **Remoción de Obstáculos:** Gestionar cualquier bloqueo técnico o administrativo que detenga al equipo de desarrollo.
3. **Coaching:** Fomentar la autogestión y el uso de "preguntas poderosas" para la resolución de problemas internos.
4. **Protección:** Blindar al equipo de cambios de alcance no planificados durante el Sprint.

## Lo que este rol NO hace
* No asigna tareas individuales (el equipo se autogestiona).
* No es una secretaría administrativa (no se limita a tomar notas).
